Driver for estimating earthquake magnitude from peak ground displacement.

Parameters
[in]pgd_propsPGD inversion parameters
[in]SA_latevent latitude (degrees) [-90,90]
[in]SA_lonevent longitude (degrees) [0,360]
[in]SA_depevent depth (km)
[in]pgd_datadata structure holding the peak ground displacement data, weights, and logical mask indicating site is a candidate for inversion
[out]pgdresults of the PGD estimation grid-search, the variance reduction at each depth in the grid search, and the sites used in the inversion
Returns
0 indicates success 1 indicates an error on the input pgd structure 2 indicates error in input pgd_data structure 3 indicates insufficient data for inversion 4 indicates an error in computation
