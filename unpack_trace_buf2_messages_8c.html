Unpacks the tracebuf2 messages read from the ring and returns the concatenated data for the desired SNCL's in the tb2Data struct.

Parameters
[in]nReadnumber of traces read off the ring
[in]msgstracebuf2 messages read off ring. the i'th message is given by (i-1)*MAX_TRACEBUF_SIZ for i=1,2,...,nRead
[in,out]tb2Dataon input contains the desired SNCL's whose data will be unpacked from the header (should it be present). on output contains the data in the messages for the SNCL's defined in this structure (should a match be found).
Returns
0 indicates success
