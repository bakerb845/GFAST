Writes a segment (fault patch) of the finite fault. A segment is comprised of vertices and slip.

Parameters
[in]shapeSegment shape. This can be: LINE (2) which has two points. TRIANGLE (3) which has three points. RECTANGLE (4) which has four points.
[in]latsthe latitudes at each vertex. the length should correspond to the number of points indicated by shape.
[in]lat_unitsunits for the input latitudes (e.g. DEGREES)
[in]lonsthe longitudes at each vertex. the length should correspond to the number of points indicated by shape.
[in]lon_unitsunits for the input longitudes (e.g. DEGREES)
[in]depthsthe depths at each vertex. the length should correspond to the number of points indicated by shape.
[in]depth_unitsunits for the input depths (e.g. KILOMETERS)
[in]ssslip in strike
[in]ss_unitsunits for slip in strike direction on segment (e.g. METERS)
[in]dsslip in dip direction
[in]ds_unitsunits for slip in dip direction on segment (e.g. METERS)
[in]ss_unceruncertainty on slip in strike direction for this segment with units given by ss_unc_units. if negative this value is ignored.
[in]ss_uncer_unitsunits for uncertainty for slip in strike direction (e.g. METERS)
[in]ds_unceruncertainty on slip in strike direction for this segment with units given by ds_unc_units. if negative this value is ignored.
[in]ds_uncer_unitsunits for ucnertainty for slip in dip direction (e.g. METERS)
[in,out]xml_writerxmlTextWriterPtr to which to write segment
Returns
0 indicates success
