Writes the focal mechanism and derived quantities such as the scalar moment, double couple and CLVD percentage, nodal planes, and principal axes to QuakeML.

Parameters
[in]publicIDrootQuakeML public ID root (e.g. quakeml:us.anss.org/).
[in]evidEvent ID.
[in]methodMethod by which the focal mechanism was computed (for this project use gps).
[in]mtMoment tensor in NED format with units of Newton-meters. The moment tensor is packed .
[in,out]xml_writerOn input this is a pointer to the xmlTextWriterPtr. On successful output the focal mechanism has been appended to the xml_writer as a QuakeML focal mechanism.
Returns
0 indicates success.
