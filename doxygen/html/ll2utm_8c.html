This takes lat and lon values and converts to UTM northing and easting.

Parameters
[in]lon_degLongitude to convert to UTM easting (degrees).
[in]lat_degLatitude to convert to UTM northing (degrees).
[out]UTMNorthingCorresponding northing UTM coordinate (m).
[out]UTMEastingCorresponding easting UTM coordinate (m).
[out]lnorthpIf true then this UTM point is in the northern hemisphere.
If false then this UTM point is in the southern hemisphere.
[in,out]zoneIf ==-1 then choose the central meridian from the input longitude.
Otherwise set the desired zone in range [0,60] from the input longitude.
