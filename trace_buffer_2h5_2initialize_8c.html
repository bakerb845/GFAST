Initializes the HDF5 file for archiving the acquisition or reading in playback mode.

Parameters
[in]jobIf job = 1 then the file will be opened as read only. If job = 2 then the file will be opened as read/write.
[in]linMemoryIf true then keep the opened file in memory.
[in]h5dirdirectory where HDF5 file exists
[in]h5filename of the HDF5 file in the directory
[in,out]h5traceBufferOn input, contains the names of the HDF5 dataset names. On output, contains the HDF5 file handle from which we'll read and write.
Returns
0 indicates success
