Writes the moment tensor, scalar moment, double couple percentage, and CLVD percentage.

Parameters
[in]publicIDrootQuakeML public ID root (e.g. quakeml:us.anss.org/)
[in]evidEvent ID.
[in]methodMethod by which the moment tensor was computed (for this project use gps).
[in]M_useThe moment tensor terms (Nm) in USE coordinates packed: .
[in]M0Scalar moment (Nm).
[in]dc_pctPercent double couple [0,100].
[in]clvd_pctPercent CLVD [0,100].
[in,out]xml_writerOn input this is a pointer to the xmlTextWriterPtr. On successful output the moment tensor has been appended to the xml_writer as a QuakeML moment tensor.
Returns
0 indicates success
