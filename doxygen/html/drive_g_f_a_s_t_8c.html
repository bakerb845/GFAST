Expert earthquake early warning GFAST driver.

Parameters
[in]currentTimeCurrent epochal time (UTC seconds)
[in]propsHolds the GFAST properties.
[in,out]eventsThe input event list. If there are no events this function will immediately return. On output, if an event has expired then it will be popped from the list.
[in,out]gps_dataHolds the GPS streams to be used in the inversions.
[in,out]h5traceBufferHolds the requisite information for reading.
[in,out]pgd_dataWorkspace for the PGD data in the PGD inversion.
[in,out]cmt_dataWorkspace for the offset data in the CMT inversion.
[in,out]ff_dataWorkspace for the offset data in the finite fault inversion.
[in,out]pgdWorkspace for the PGD inversion.
[in,out]cmtWorkspace for the CMT inversion.
[in,out]ffWorkspace for the finite fault inversion.
[out]xmlMessagesContains the XML messages for all events for the PGD and finite fault for activeMQ to forward onto shakeAlert as well as the CMT quakeML.
Returns
0 indicates success.
