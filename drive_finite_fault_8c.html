Drives the finite fault fault plane grid search inversion.

Parameters
[in]ff_propsfinite fault inversion parameters
[in]SA_latevent latitude (degrees)
[in]SA_lonevent longitude (degrees)
[in]SA_depevent depth (km). note, it may be advantageous to obtain this from the CMT inversion
[in]ff_dataoffset data for the finite fault inversion
[in,out]ffon input contains space for the finite fault inversion as well as the fault strike and rake on all the nodal planes in the inversion. on output contains the results of the finite fault inversion which includes slip along strike and slip down dip on each plane, uncertainties, and estimate data and observed data
Returns
0 indicates success.
