Copies the HDF5 trace buffer data onto the GPS data structure.

Parameters
[in]traceBuffercontains data from HDF5 file to copy onto the GPS data structure. Notice, the trace buffer is appropriately queried for times [t1, t2]
[in,out]gps_dataon input contains the requisite space to store the data from the traceBuffer. on output contains the data from [t1, t2] and fills all excess points (up to maxpts) with NAN.
Returns
0 indicates success
