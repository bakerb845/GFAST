Convenience function to pop expired events from the events list. An event is expired when: currentTime - eventOriginTime > processingTime.

Parameters
[in]processingTimeMax processing time (seconds) for any event.
[in]currentTimeCurrent time (UTC epochal seconds).
[in]verboseControls the verbosity. 0 is quiet.
[in,out]eventsOn input holds the active events list. On output expired events have been removed.
Returns
The number of events removed from the list.
