Parameters

[in,out]ffon input contains space for the finite fault inversion as well as the fault strike and rake on all the nodal planes, and centroid depth in the inversion. on output contains the results of the finite fault inversion which includes slip along strike and slip down dip on each plane, uncertainties, and estimate data and observed data
