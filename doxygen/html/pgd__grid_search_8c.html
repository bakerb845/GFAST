PGD scaling full grid search (lat, lon, depth) driver.

Parameters
[in]l1Number of sites.
[in]ndepsNumber of source depths.
[in]verboseControls verbosity (< 2 is quiet).
[in]dist_tolDisplacement tolerance (cm). If the displacment is less than dist_tol it will be set to dist_tol.
[in]disp_defDisplacement default (cm) if d < dist_tol.
[in]utmSrcNorthingsSource UTM northing positions (m). This is an array of dimension [nlat*nlon] with leading dimension nlats.
[in]utmSrcEastingsSource UTM easting positions (m). This is an array of dimension [nlat*nlon] with leading dimension nlats..
[in]srcDepthsSource depth in grid search (km). This is an array of dimension [ndeps].
[in]utmRecvNorthingReceiver UTM northing position (m). This is an array of dimension [l1].
[in]utmRecvEastingReceiver UTM easting position (m). This is an array of dimension [l1].
[in]staAltStation elevation (m). This is an array of dimension [l1].
[in]dSite peak ground displacements (cm). This is an array of dimension [l1].
[in]wtsData weights on each observation. Nominally, this is an array of dimension [l1]. However, if NULL or if each weight is the same then this array will be ignored.
[out]srdistSource receiver distance between the idep'th source and k'th receiver. This is an array of length [l1*ndeps*nlat*nlon]. The i'th site at the idep'th depth is given by ilon*nlats*ndeps*l1 + ilat*ndeps*l1 + idep*l1 + i
[out]MMagnitude at each depth. This is an array of length [ndeps*nlat*nlon]. The (ilat,ilon,idep)'th is given by ilon*nlats*ndeps + ilat*ndeps + idep.
[out]VRVariance reduction (percentage) at each depth. This is an array of length [ndeps*nlat*nlon]. The (ilat,ilon,idep)'th is given by ilon*nlats*ndeps + ilat*ndeps + idep.
[out]iqrThe interquartile range computed from the difference of the 75th percentile of the weighted residuals and the 25th percentile of the weighted residuals at each depth [ndeps]. The (ilat,ilon,idepth)idep'th is given by ilon*nlats*ndeps + ilat*ndeps + idep.
[out]UestThe PGD estimate peak ground displacements. This is an array of dimension [nlons*nlats*ndeps*l1]. The i'th estimate at the idep'th depth is given by ilon*nlats*ndeps*l1 + ilat*ndeps*l1 + idep*l1 + i.
Returns
0 indicates success.
